# gb-music-vue

## 一个基于 vue 和 express 和 mySQL 的音乐后台管理系统

```
音乐后台管理系统主要实现歌曲，歌手，专辑的添加和删除以及信息的修改
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
